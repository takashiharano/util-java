# Util
Various functions for performing general functions.

## Features
- String
- Date/Time
- File
- HTTP
- Base64
- Hash
- Randomizer
- Ring Buffer
- Binary Operation
- Hex Dumper
- CSV Builder / Parser
- JSON Builder / Parser
- CommandExecution
- Props
- String Permutation
- Color Manipulation
- Logger
- ZIP

## Build

```sh
mvn clean package
```
